## Security Policy

### Supported versions

This is a research/tooling repository. Security fixes may be released without backporting.

### Reporting a vulnerability

If you believe you found a security issue (e.g., path traversal, unsafe extraction behavior), please **do not** open a public issue.

Instead, contact the maintainer and include:
- A clear description of the impact
- Steps to reproduce
- A minimal sample (if possible) or hexdump / metadata needed to reproduce

We will acknowledge the report and respond as soon as possible.
